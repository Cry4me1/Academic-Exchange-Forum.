🎨 Scholarly UI 创新更新 — 构思方向
基于你现在的项目结构（三栏式 Dashboard、帖子详情页、VIP/积分系统、个人主页、决斗场等），我梳理出 5 个可以做的创新方向：

方向 A · 🏠 Dashboard 信息流重构
当前 Dashboard 是经典三栏布局（侧边导航 + 帖子流 + 右侧公告）。可以升级为：

Pinterest/瀑布流卡片布局，帖子卡片高度不等，更有视觉冲击力
帖子卡片增加封面图预览、标签色带、作者 VIP 闪光效果
信息流顶部增加「故事条」（类似 Instagram Stories），展示好友/热门用户最新动态
方向 B · 📝 帖子详情页阅读体验升级
引入沉浸式阅读模式（一键隐藏侧栏，文章居中放大，加上柔和背景光）
浮动目录导航 (TOC) 固定在侧边，滚动跟随高亮
评论区重新设计：引入类似 GitHub Discussions 的投票排序、折叠回复、评论高亮
方向 C · 🎮 决斗场 & 社交互动动效升级
决斗对战时增加实时倒计时动画、VS 分屏特效
好友列表增加在线状态指示器（绿色呼吸灯）
全局增加成就弹窗 / 徽章展示系统（首次发帖、首次 AI 调用、连续签到等）
方向 D · 🌙 全站主题系统升级
从目前的 light 模式扩展为 Dark Mode / System Auto 三档切换
增加全站配色方案（不仅是个人主页背景，全站都能选配色）
导航栏、卡片、按钮都随主题变化，添加流畅的切换过渡动画
方向 E · 📱 移动端体验 & 微交互打磨
底部 Tab 导航栏（移动端专属），替代汉堡菜单
Pull-to-refresh 下拉刷新动画
帖子点赞/收藏的微动画升级（心跳爆炸、星星粒子）
长按帖子卡片出现快捷操作浮层
