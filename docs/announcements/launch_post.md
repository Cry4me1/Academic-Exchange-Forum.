# 致 Scholarly 的首批用户：欢迎来到学术交流的新天地

大家好！

今天，我们非常激动地宣布，**Scholarly** —— 这个专为研究人员、学生和学术爱好者打造的现代化交流社区，正式上线了！

在信息爆炸的今天，找到一个既能严肃讨论学术问题，又能享受现代化交互体验的平台并不容易。很多时候，我们在陈旧的论坛界面中挣扎，或者在泛娱乐化的社交媒体中迷失。Scholarly 的诞生，正是为了填补这一空白。

## 我们带来了什么？

### 1. 专为学术打造的沉浸式编辑器 ✍️
我们深知，学术讨论离不开公式、代码和图表。Scholarly 集成了基于 `Novel` 的现代化富文本编辑器，为您提供：
- **完美的 LaTeX 支持**：无论是行内公式 `$E=mc^2$` 还是复杂的块级公式，都能优雅渲染。
- **智能的 Slash 命令**：只需输入 `/`，即可快速插入标题、列表、引用、代码块等。
- **语法高亮**：对各类编程语言提供一流的语法高亮支持。

### 2. 简洁而强大的现代化 UI 🎨
告别上个世纪的论坛界面。我们采用了当今最前沿的 **Next.js** 技术栈，配合 **Shadcn/UI** 和 **Tailwind CSS**，为您呈现：
- **响应式设计**：无论是在宽屏显示器还是移动设备上，都能获得流畅的阅读体验。
- **暗色模式**：深夜不论是赶论文还是刷论坛，都能呵护您的双眼。

### 3. 实时互动的社区体验 💬
学术灵感往往诞生于思维的碰撞。
- **实时通知**：当有人回复您的帖子或点赞时，您会立即收到通知。
- **即时通讯**：支持私信功能，方便进行深度的学术切磋。

### 4. 数据安全与隐私 🔒
基于 **Supabase** 构建的后端架构，利用 Row Level Security (RLS) 技术，确保您的数据只有您自己和授权用户可见。

## 📖 编辑器使用教程：三步玩转 Scholarly

为了让您获得最佳的写作体验，我们准备了这份简易教程。

### 第一步：万能的 Slash 命令 (`/`)
在任意空行输入 `/` (斜杠)，即可唤起我们的增强命令菜单。您不仅可以快速设置标题，还能：
- **📝 列表控制**：快速插入无序列表、有序列表，甚至是 **待办事项 (Todo List)** 来管理任务。
- **💻 代码块**：输入 `/code` 插入代码块，我们将自动识别语言并高亮。
- **❝ 引用**：使用 `/quote` 插入高亮的引用块。

### 第二步：享受数学的乐趣 (`LaTeX`)
我们为理工科用户深度定制了数学公式支持：
- **行内公式**：像这样输入 `$E=mc^2$`，即可渲染为优雅的数学符号。
- **公式命令**：输入 `/math` 快速插入公式模板。
- **复杂公式**：支持标准的 LaTeX 语法，无论是矩阵还是积分，都能完美呈现。

### 第三步：图文并茂
- **快捷上传**：输入 `/image` 唤起上传器。
- **拖拽交互**：直接将桌面上的图片拖入编辑器位置即可上传（支持最大 2MB 图片）。
- **Notion 风格**：将鼠标悬停在段落左侧，按住 `::` 按钮即可拖拽段落重新排序。

## 给我们一点时间

Scholarly 目前处于 **Beta** 阶段。这意味着它可能还不够完美，您在使用过程中可能会遇到一些 bug 或体验不顺畅的地方。

如果您发现了问题，或者有任何改进建议，请直接在社区的 [反馈专区] 发帖，或通过私信联系管理员。您的每一条反馈，都是帮助 Scholarly 成长的养分。
