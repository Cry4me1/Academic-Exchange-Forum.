# 📝 发帖编辑器指南

Scholarly 的编辑器专为学术交流打造，融合了 Notion 风格的流畅体验与专业的学术功能支持。以下是核心功能介绍。

1. 基础操作 (Basics)

快速上手您的创作工具：

- **Slash 命令**: 在空白行输入 `/` (斜杠)，即可唤起菜单快速插入标题、列表、引用等。
- **Markdown 支持**: 完美兼容标准语法，如使用 `#` 创建标题，`**` 加粗文字。
- **快捷键**: 支持通用的编辑快捷键，提升写作效率。

2. 数学公式 (Mathematics)

为理工科交流提供完美支持：

- **行内公式 (Inline)**: 使用 `$` 包裹 LaTeX 代码。例如 `$E=mc^2$` 会显示为 $E=mc^2$。
- **块级公式 (Block)**: 输入 `$$` 或使用 `/math` 命令，适合展示复杂的公式推导或矩阵。

3. 代码高亮 (Code)

清晰展示您的算法实现的细节：

- **插入代码**: 输入 \`\`\` (三个反引号) 后跟语言名 (如 `python`)，或使用 `/code` 命令。
- **语法高亮**: 支持多种主流编程语言的高亮显示，提升阅读体验。

4. 媒体与交互 (Media & Interaction)

打造图文并茂的学术文章：

- **图片上传**: 输入 `/image` 或直接将图片拖入编辑器（支持最大 2MB 文件）。
- **拖拽排版**: 鼠标悬停在段落左侧出现的 `::` 图标上，按住即可自由拖拽移动段落位置。

如果有任何疑问，欢迎直接在帖子下方留言。
