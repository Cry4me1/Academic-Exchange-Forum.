# Scholarly 使用指南 & 编辑器手册

欢迎来到 Scholarly！为了帮助您更好地使用本平台进行学术交流，我们准备了这份详细的操作指南，特别是关于如何使用我们要大的富文本编辑器。

## 1. 账号与登录 🔐

Scholarly 提供多种便捷的登录方式：
- **邮箱登录**：传统的邮箱+密码组合。
- **Magic Link**：如果您忘记密码或不想记忆密码，直接发送登录链接到邮箱，点击即登录。

登陆后，别忘了去 **个人中心** 设置您的头像和个性签名，向大家介绍您的研究方向！

## 2. 发帖编辑器全攻略 📝

这是 Scholarly 最核心的功能。我们的编辑器并非普通的文本框，而是一个功能强大的创作工具。

### ⚡ 核心操作：Slash 命令 (`/`)

在编辑器任意新行的开头输入 `/`（斜杠），会弹出一个命令菜单。通过它，您可以快速插入：
- 各级标题（H1, H2, H3）
- 无序/有序列表
- 引用块
- **代码块**
- **数学公式**
- 图片上传

### 📐 数学公式 (LaTeX)

我们对数学公式提供了原生的一流支持。

**行内公式**：
使用美元符号包裹 LaTeX 代码。
输入：`我们知道 $E = mc^2$ 是质能方程`
显示：我们知道 $E = mc^2$ 是质能方程

**块级公式**：
输入 `$$` 然后按回车，或者使用 `/math` 命令。
输入：
```latex
\sum_{i=1}^{n} i = \frac{n(n+1)}{2}
```
将会显示为居中的大公式。

### 💻 代码块

输入 \`\`\` (三个反引号) 后跟语言名称（如 `python`），或者使用 `/code` 命令。
编辑器会自动识别语言并进行语法高亮。

```python
def hello_world():
    print("Hello Scholarly!")
```

### 🖼️ 图片上传

- **拖拽上传**：直接将电脑里的图片拖入编辑器即可。
- **粘贴上传**：截图后，在编辑器内直接 `Ctrl+V` (Cmd+V)。
- **限制**：目前支持最大 2MB 的图片文件。编辑器支持简单的图片缩放功能。

### 🪄 AI 辅助 (即将上线)

我们正在测试 AI 续写功能，未来您可以通过点击 "++" 按钮请求 AI 帮助润色或扩展您的学术观点。

## 3. 互动与社交 🤝

- **评论**：支持富文本评论（同样支持公式！）。
- **点赞/收藏**：遇到高质量的 Paper 分享或技术贴，记得收藏以防走丢。
- **私信**：点击用户头像进入主页，可发起私聊。

## 4. 常见问题 (Q&A)

**Q: 发出的帖子格式乱了怎么办？**
A: Scholarly 的编辑器是 "所见即所得" (WYSIWYG) 的，但如果 Markdown 解析出现问题，请尝试删除格式错误的段落重新使用 `/` 能够插入。

**Q: 为什么上传图片失败？**
A: 请检查图片是否超过 2MB。如果网络不稳定，也可能导致上传失败。
