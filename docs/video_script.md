## 💡 录制小贴士 (Pro Tips)

1.  **分辨率**: 尽量使用 1920x1080 (1080p) 或 4K 录制，保证文字清晰。
2.  **清理环境**: 
    - 隐藏浏览器书签栏 (`Ctrl+Shift+B` / `Cmd+Shift+B`)。
    - 关闭所有无关的浏览器插件图标。
    - 使用浏览器的“隐身模式”或新建一个干净的 Profile。
3.  **平滑操作**:
    - 鼠标移动不要太快，点击要准确。
    - 滚动页面时使用鼠标中键或触控板的平滑滚动，不要直接拖动滚动条（太生硬）。
4.  **Zoom In**: 在剪辑时，当展示代码细节或公式时，运用**局部放大** (Zoom) 效果，不仅能看清细节，还能增加视频的动感。
